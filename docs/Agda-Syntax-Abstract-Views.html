type AppView = AppView' Expr Source #

appView :: Expr -> AppView Source #

Gather applications to expose head and spine.

Note:

  1. Everything is an application, possibly of itself to 0 arguments.
  2. If the returned head is a projection originating from a post-fix projection, the ArgInfo of the first argument might be incorrect. (See issue #8300.) For instance x .f is valid postfix syntax for f {{x}}, but appView turns the former into f x. The correct f {{x}} is produced by appViewM.
