Agda.Interaction.Library.Parse

Description

Parser for .agda-lib files.

Example file:

    name: Main
    depend:
      standard-library
    include: .
      src more-src

Should parse as:

    AgdaLib
      { libName     = Main
      , libFile     = path_to_this_file
      , libIncludes = [ "." , "src" , "more-src" ]
      , libDepends  = [ "standard-library" ]
      }
