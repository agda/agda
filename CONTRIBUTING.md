# Contributing

Thank you for your interest in contributing to Agda. To get you started,
we collect notes on navigating the codebase in
[`HACKING.md`](https://github.com/agda/agda/blob/master/HACKING.md), and
[issues that would make for good first
contributions](https://github.com/agda/agda/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22difficulty%3A%20easy%22),
as well as [issues for which we would welcome pull
requests](https://github.com/agda/agda/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22PR%20welcome%22).

## Use of generative AI

Agda does not accept contributions generated by large language models.
This includes pull requests and issue reports. This includes code and
reports generated by, or with the use of, tools such as Claude (incl.
Claude Code), ChatGPT, DeepSeek, and Copilot.
